# Blog Application Readme

Welcome to the Blog Application! This application allows users to create, manage, and share their blog posts. This README file provides an overview of the application's modules and instructions on how to run it.

## Modules

1. **Register**
   - New users can register for an account by providing their email and a secure password.

2. **Login**
   - Registered users can log in to their accounts using their email and password.

3. **Account Settings**
   - Users can manage their account settings, including updating their profile information and changing their password.

4. **Create New Post**
   - Users can create new blog posts by providing a title, content, and optionally adding images or other media.

5. **Watch All Posts**
   - Users can view a list of all the blog posts available on the platform, including their own posts and posts from other users.

6. **Update/Delete Post**
   - Users can edit and update their existing blog posts, as well as delete them if needed.

## Running the Application

To run the Blog Application, you'll need both the frontend and backend components up and running.

### Frontend

Follow these steps to run the frontend part of the application:

1. Open a terminal window and navigate to the root directory of the frontend project.

2. Run the following command to install the required dependencies:

   ```bash
   npm install
  
3. After the installation is complete, start the frontend server by running:
     ```bash
      npm start
The frontend should now be accessible in your web browser at http://localhost:3000.

### Backend

The backend part of the Blog Application is responsible for handling API requests and interacting with the database (MongoDB). Follow these steps to run the backend:

Ensure you have MongoDB installed and running on your system. You can download MongoDB from https://www.mongodb.com/try/download/community and follow the installation instructions.

Open a terminal window and navigate to the root directory of the backend project.

Run the following command to install the required Node.js dependencies:

    
        npm install

After the installation is complete, start the backend server by running:
 
    npm start
The backend server should now be running on http://localhost:5000.

Now, you should have both the frontend and backend of the Blog Application up and running. You can access the application by opening your web browser and navigating to http://localhost:3000.

Enjoy using the Blog Application! If you encounter any issues or have questions, please refer to the documentation or contact our support team.
